Pastebin - back

Goal

Use the Pastebin API to send a snippet of text/code to be hosted by Pastebin

Block the submit if the form is empty; otherwise, send the data to Pastebin, and display the returned link.

To do

  1. Listen the submit event of the form ( help)
  2. Prevent the refresh of the page with PreventDefault() (help)
  3. Keep the value of filename, expiration & code in the state
  4. On submit, check if any property of the state (filename, expiration & code) is empty, if not submit the form
  5. If one element is empty, display a message
  6. Replace ~~~~ by the generated link given by the API

On this exercise, you can use Fetch or Axios.

You can find the API documentation here: https://pastebin.com/doc_api

Note1: You DO NOT to create an API key with Pastebin to solve this exercise. If you look inside App.js; the key is already provided to you under the <input> with a name of api_dev_key; alongside with other optional parameters you can use with the API (like api_paste_private). How do you get access to those though?

Note2: this API has limitations. To go around those, and only for the purpose of this exercise, we use a CORS proxy to bypass those limitations Click here and click on the Request temporary access to the demo server button to be able to use the API.
